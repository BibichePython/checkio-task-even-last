Дан массив целых чисел. Вам нужно найти сумму элементов с четными индексами (0ой, 2ой, 4ый итд). Затем перемножить эту сумму и последний элемент исходного массива. Не забудьте, что первый элемент массива имеет индекс 0.

Для пустого массива, результат всегда 0 (ноль).

Вх. данные: Список (list) целых чисел (int).

Вых. данные: Число как целочисленное (int).

Примеры: 

checkio([0, 1, 2, 3, 4, 5]) == 30
checkio([1, 3, 5]) == 30
checkio([6]) == 36
